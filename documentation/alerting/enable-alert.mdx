This article explains how to re-activate a alert in Monita. Alerts are used to create notifications which can alert you when specific scenarios occur over a set period.

<Note>Once alerts are Active you will begin receiving Slack/Email notifications if the alert condition is met.</Note>

### Instructions
1. Go to alerts page
2. Press the toggle switch on any alert you wish to Activate
![Toggle alert activation](/images/alerting/alert-activation-off.png)
3. A toast notification will confirm the alert has been Activated 
![Alert activation confirmed](/images/alerting/alert-activation-confirmation.png)
