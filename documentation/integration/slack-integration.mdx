### Instructions
1. [Login](https://app.getmonita.io) to the app
2. Go to [Integrations](https://app.getmonita.io/integrations)
3. Click 'New Integration'
![](/images/integration/integration-new-integration-button.png)
4. Select Slack and press 'Connect with Google'
![](/images/integration/integration-select-slack.png)
5. Allow Slack to integrate with a workspace and select a channel which the App will post to
![](/images/integration/integration-select-slack-workspace.png)
6. Press 'Allow'
7. If integration is successful, a tile will appear with a green status light. 
![](/images/integration/integration-slack-integeration-card.png)
8. (Optional) Select a default workspace. This will speed up alert ceration by negating the need to select a slack channel for alerts destination
![](/images/integration/integration-slack-integeration-card-default-channel.png)
<Note>After connecting it may take a few minutes for the changes in the Slack workspace to take effect.</Note>
