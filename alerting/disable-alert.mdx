This article explains how to de-activate a alert in Monita. Alerts are used to create notifications which can alert you when specific scenarios occur over a set period.

<Note>Once alerts are deactivated you will no longer receive Slack/Email notifications. If you would like to stop receive a notification for a single tag, instead Update the alert and by removing the tag reference from the alert.</Note>

### Instructions
1. Go to alerts page
2. Press the toggle switch on any alert you wish to deactivate 
![Toggle alert activation](/images/alerting/alert-activation-on.png)
3. A toast notification will confirm the alert has been deactivated 
![Alert activation confirmed](/images/alerting/alert-deactivation-confirmation.png)
