### Instructions

To monitor a new tag follow these instructions:

1. Go to the [Monitoring home page](https://beta.raptor.digital/monitoring)
2. Click "Manage" on a container you would like to update
3. **Press the Refresh icon** on the right of the search tags field
![Search Tags](/images/monitoring/gtm-monitoring-search-tags.png)
4. Once tags are synced the new ones will appear below
![Tag selection](/images/monitoring/gtm-monitoring-tag-selection.png)
5. Select the new tag and press Save
6. Remember to submit the changes in Google Tag Manager
